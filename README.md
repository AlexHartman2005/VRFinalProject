This is my final project for CIS442.
It is a simple farming game
My goals are this:

Users should be able to start a farm and grow a few different plants. (plant a seed, water it, make sure it has sunlight, harvest when it is fully grown.
Their goal is to escape the city by making enough money by selling crops to get a ticket on the bus to the city

Map: it will be a cute little farm the players will work inside
Controller: it will be hands, so that you can grab seeds, use a scythe, etc.
Interaction: You will be able to select things like the seeds, the water bucket, and interact with the scythe
Environmental Tracking - Collision/raycasting: They scythe will need to collide with the wheat (or whatever you plant) to haverst it. Obviously, if you swing the tool and don't hit it, nothing will happen. If you do hit it, (which will be detected with collision) it will destroy and give you the crop
End point: Make enough money to take the bus to leave the farm
